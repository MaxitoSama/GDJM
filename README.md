## Information about the game.
Ninja's Path

Short 2D platform game with enemies that make pathfinding.
There are two types of controls.

hola això és una prova

For gameplay:
 - Left  			= "D"
 - Right 			= "A"
 - Slide 			= "S + A/D"
 - Jump  			= "Space"
 - Sprint			= "A/D + Left Shift"
 - Music Volume Up 		= "+"
 - Music Volume Down 		= "-"
 - Save  			= "F5"
 - Load				= "F6"

For Testing:
 - First Scene     		= "F1"
 - Start of Scene 		= "F2"
 - See Colliders 		= "F3"
 - Create Path			= "F8"
 - Save				= "F5"
 - Load				= "F6"
 - God Mode			= "F10"
